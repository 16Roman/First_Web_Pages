Important Principles to Keep in Mind when Developing for Mobile Devices

    Responsive Design Choose a Flex or Grid Layout Media Queries

Responsive Design

It is important to make sure images and text still appear as intended, even if the user is viewing the website on a smaller screen

Some tools you can use to make your website more responsive include:

  • %
  • rem

The "%" make the image or text take up a certain percentage of the page. This makes it so no matter the size of the webpage, the picture will take up the same amount of space.

The "rem" dictates an element's font size relative to the size of the root element. It can be easily doubled with "2rm"

Choose a Flex or Grid Layout

Depending on how you want to organize your website, you will be using a Flex or Grid layout.

When Should I Use a Flex box Layout?

Flex boxes are laid out in one dierction, but can be changed to become a column.

When Should I Use a Grid Layout?

The grid layout divides the wesite into rows and columns

Media Queries

Media Queries allows you to have different website layouts depending on the size of the screen.

You will be able to have a website with a distinct desktop and mobile interface. You don't have to create one website layout that will be perfect for both mobile and dsktop users.

You can use media queries to make any necessary changes to your website so it will work well on all devices.
